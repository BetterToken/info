# Titan Node
The Titan Node is a medium sized node that already packs quite a punch and will be able to run heavier workloads.


## Specifications

* AMD Ryzen 5 3400G
* 32 GB Ram
* 512 GB SSD (NAND)
* 1 TB HDD

## Dimensions

* Width 176 mm
* Height 200 mm
* Depth 380 mm
