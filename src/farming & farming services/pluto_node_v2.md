# Titan Node
The Titan Node is a medium sized node that already packs quite a punch and will be able to run heavier workloads.


## Specifications

* AMD Ryzen 3 3200G
* 8 GB Ram
* 480 GB SSD 

## Dimensions

* Width 176 mm
* Height 200 mm
* Depth 380 mm
